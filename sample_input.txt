# Jenil Demo

### Demo Overview

- Jenil presented machine learning project on customer segmentation
- Dataset: E-commerce transactions with invoice numbers, stock codes, quantities, dates, prices, customer IDs, countries
- Goal: Predict customer behavior and purchasing patterns for business recommendations

### Data Processing & Cleaning

- Initial dataset: 541,909 records with 51,000+ unique customers
- Key issues identified:
  - Missing values in customer IDs (1,454 null values)
  - Negative quantities indicating returns/cancellations
  - Zero unit prices requiring investigation
- Data transformation steps:
  - Created total sales column (quantity × unit price)
  - Merged transaction data with last transaction dates
  - Removed outliers using interquartile range method
  - Applied mean scaling for normalization

### Machine Learning Implementation

- Used unsupervised clustering algorithm for customer segmentation
- Applied elbow method to determine optimal number of clusters (3 clusters identified)
- Model accuracy achieved: 0.78 (78%)
- Visualization through scatter plots and heat maps for correlation analysis

### Real-World Applications

- E-commerce website product recommendations
- Customer behavior prediction for targeted marketing
- Inventory management based on purchasing patterns
- Discount strategy optimization for different customer segments
